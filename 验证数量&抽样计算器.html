置信度选用建议

90% 置信度

100次验证中约90次结论可靠。适用于内部流程监控、非关键产品检验、快速抽样排查。样本量最小，检验成本低。

95% 置信度（推荐）

100次验证中约95次结论可靠。适用于常规产品出厂检验、供应商来料检验、批量生产质量控制。兼顾可靠性与成本。

99% 置信度

100次验证中约99次结论可靠。适用于高风险产品、医疗/汽车关键部件、客户严格质量验证。结论可信度最高。

计算公式

正向计算（验证数量）

核心公式

n = ⌈ ln(1 − C) / ln(1 − p₀) ⌉

  • C = 置信度，p₀ = 预估不良率
  • ⌈·⌉ = 向上取整
  • 基于 Poisson 分布近似的零缺陷验证模型

反向计算（最大不良率）

核心公式

p_max = 1 − (1 − C)^(1/n)

  • C = 置信度，n = 抽检数量
  • α = 1 − C = 显著性水平
  • 等价形式：p_max = 1 − α^(1/n)

零缺陷验证模型介绍

一、模型核心

零缺陷验证模型（"零接收抽样方案"）核心目标：抽取 n 个样本检验，若未发现不良品，则在置信度 C 下确认总体不良率不超过 p₀。

应用场景：产品出厂验证、流程稳定性测试、供应商来料检验、关键工序管控。

二、统计原理

设总体不良率为 p，样本中无不良品的概率：

P(无不良) = (1 − p)^n

要求：P(p ≤ p₀ | 样本无不良) ≥ C，通过贝叶斯公式推导可得上述公式。

三、适用范围与局限性

适用场景

  • 预估不良率较低（p₀ ≤ 10%）的场景
  • 仅关注"是否存在不良"的定性验证
  • 批量生产的规模化质量检验

局限性

  • 若样本中发现不良品，需扩大样本量
  • 高不良率（p₀ > 10%）误差较大
  • 未考虑漏检、误检的影响

适用场景：产品不良率验证、流程稳定性测试、质量控制等计数型数据场景
